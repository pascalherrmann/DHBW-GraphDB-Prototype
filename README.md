# DHBW-GraphDB-Prototype

### Getting Started with Docker

##### PHP + Neo4j + Arangodb:
```
docker-compose up
```
* Go To http://localhost:8081

##### Node.js + Neo4j:
```
docker-compose -f docker-compose-node.yaml up
```
* Go to http://localhost:8080
